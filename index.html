Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability.

Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages worldwide.

Key features of Python include:

  • Easy-to-learn syntax
  • Cross-platform compatibility
  • Extensive standard library
  • Support for multiple programming paradigms
print("Hello, World!")
Getting Started

To start programming in Python, you'll need to install the Python interpreter on your computer.

Here are the basic steps to get started:

  1. Download Python from the official website
  2. Run the installer
  3. Verify the installation

After installation, you can run Python code in several ways:

>>> python --version Python 3.9.0

You can use the interactive shell, write scripts, or use an IDE like PyCharm or VS Code.

Variables and Data Types

Python has several built-in data types:

  • Numbers (int, float, complex)
  • Strings
  • Booleans
  • Lists
  • Tuples
  • Dictionaries

Variables are created when you assign a value to them:

x = 5 # int y = "Hello" # str z = [1, 2, 3] # list

Python is dynamically typed, so you don't need to declare variable types.

Control Flow

Python supports the usual control flow statements found in other languages:

if x < 0: print("Negative") elif x == 0: print("Zero") else: print("Positive")

Looping in Python can be done with for and while statements:

for i in range(5): print(i) while x > 0: print(x) x -= 1

Python also supports break, continue, and else clauses in loops.

Functions

Functions are defined using the def keyword:

def greet(name): """This function greets the person passed in""" print("Hello, " + name + "!") greet("Alice")

Functions can return values using the return statement:

def square(x): return x * x result = square(4) # returns 16

Python functions support:

  • Default arguments
  • Keyword arguments
  • Variable-length arguments
  • Anonymous (lambda) functions
